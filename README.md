# 📚 Book Finder App

A simple React application that allows users to search for books using the **Google Books API**.

---

## 🚀 Features
- Search for books by title or author  
- View book title, author, and description  
- Responsive and clean UI  

---

## 🛠️ Technology Stack
- React (JavaScript)
- Google Books API
- CSS for styling
